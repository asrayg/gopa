# code of conduct

gopa is for builders. be a decent human.

## the vibe

- be respectful (even when you disagree)
- assume good intent
- give clear feedback (not personal shots)
- help beginners (that's literally the point of gopa)

## unacceptable behavior

- harassment, hate speech, discrimination
- personal attacks, doxxing, threats
- spam, scams, or malicious content
- pushing unsafe code or security advice as "no big deal"

## enforcement

maintainers can delete comments, close issues, or ban users if you're making the space worse.

if something happens:

- or open a private report if the platform supports it

## scope

this applies to github issues, prs, discussions, community spaces, and any official gopa channels.

keep it chill. ship good stuff.
